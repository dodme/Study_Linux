다중사용

하나의 컴퓨터를 여러명의 사용자가 개개인의 비밀을 가지고 각자 다른 계정으로 접속해서 사용하는것
다중사용을 하면 시스템의 복잡도가 높아지는 단점이 있다


id : identify. 즉, 현재 사용자에 관한 정보를 보여준다
who : 현재 접속해있는 사용자의 목록을 보여준다


super(root) user와 일반 user의 차이점
super(root) user란 전지전능한 유저(관리자)를 말한다
user란 권한이 제한된 일반 유저들을 말한다
슈퍼유저가 아니지만 일시적으로 권한을 얻어야할떄 sudo라는 명령어를 사용한다. 하지만 모든 유저가 권한을 얻을수는 없고 슈퍼유저가 될수 있는 유저만이 sudo라는 명령어를 사용할수 있다

super유저를 구분하는 방법 : 유저의 이름이 root라는 이름을 가지고있고, 사용자 정보기호가 $가 아닌 #으로 바뀐다

super유저가 되기위해서는 su라는 명령어를 사용하면 된다.
su - 유저이름(root)를 입력하고 슈퍼유저에 대해 알고있는 비밀번호를 입력한다.
super유저에서 빠져나가고싶으면 exit를 입력


super유저가 하는 행동은 서버 전체에 영향을 끼쳐 서버 이용자들에게도 영향이 있기때문에 되도록이면 super유저로 활동하기보다는 필요할때만 sudo명령어로 권한을 받아오는걸 추천


일부 운영체제는 root사용자를 막아놨기때문에 잠겨있는 락을 풀어야한다
sudo passwd -u root 로 락을 풀수 있음(-u : -unlock). 만약 이경우 root사용자관련 패스워드가 설정되있지 않은경우 사용할 똑같은 패스워드를 두번 입력한다
다시 락을 걸고싶다면 sudo passwd -l root로 락을 걸수 있다(-l : -lock)

root사용자로 접속했을때 홈디렉토리의 위치는 /root이다
일반사용자의 홈디렉토리의 위치는 /home/사용자 이다
즉, root사용자와 일반사용자는 서로 다른 디렉토리를 사용하며 일반사용자는 root사용자의 디렉토리에 접근할수 없음


사용자를 추가하는방법
sudo useradd -m 사용자이름 : 새로운 사용자의 ID를 추가함
sudo passwd 사용자이름 : 해당 사용자의 비밀번호를 추가함
sudo usermod -a -G sudo 사용자이름 : 해당 사용자에게 superuser 사용권한을 부여함
