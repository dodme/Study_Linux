컴퓨터에서 현재 실행되고있는 프로세스를 확인하는법

ps : 프로세스 리스트를 보여주는 명령어
ps aux : 백그라운드에서 돌아가는중인 프로세스 리스트까지 모두 보는 명령어
ps aux | grap apache : 백그라운드에서 돌아가는중인 프로세스 리스트중 apache라는 값을 가진 결과만 보여주는 명령어
sudo kill 프로세스ID : 특정 프로세스를 강제로 종료시키는 명령어
sudo top : 프로세스 목록을 보여주는 명령어
sudo htop : top의 결과를 더 그래피컬하게 보여주는 명령어. 자동으로 설치가 안되어있을수도 있으니 apt-get install을 사용해서 설치해서 사용하면 됨
